
◆ 자바스크립트로 무엇을 할 수 있을까?

자바스크립트는 원래 웹 사이트에 움직이는 효과만을 주는 언어였습니다. 그래서 자바스크립트가 웹의 필수 요소는 아니었습니다.
하지만 이제는 자바스크립트로 서버를 만들거나 웹에서 작동하는 프로그램을 만드는 등 여러가지의 일을
할 수 있게 되었습니다.

 1. 웹 사이트를 동적으로 만들 수 있다.
 웹 사이트를 건물에 비유하면 HTML 은 건물의 기본 골격을 담당하고 CSS 는 건물 내외부의 인테리어를 담당합니다.
 그러나 메뉴에 마우스 포인터를 올렸을 때 하위 메뉴가 펼쳐지는 것과 같이 웹 사이트에 움직임을 주기 위해서는
 다른 언어를 사용해야 하는데, 이때 사용하는 언어가 바로 자바스크립트입니다.

 2. 웹 브라우저에서 실행되는 프로그램을 만들 수 있다.
 자바스크립트로 웹 사이트를 동적으로 만드는 것 뿐 아니라 웹에서 작동하는 프로그램도 만들 수 있습니다.
 예를 들어, 웹 사이트에서 버스 정보를 검색하는 프로그램입니다. 버스 노선이나 정류장 정보를 저장하는 기능부터
 지도에 표시하는 기능까지 모두 자바스크립트로 만들 수 있습니다.

 3. 서버를 구성하고 서버용 프로그램을 만들 수 있다.
 과거에는 서버 프로그래을 만들기 위해 PHP, ASP, .NET, JAVA 와 같은 언어가 필요했습니다.
 그러나 이제는 자바스크립트만으로도 서버 프로그램을 만들 수 있습니다. 예를 들어 Node.js 같은
 프레임워크 등을 사용하면 서버 프로그램도 쉽게 만들 수 있습니다.
