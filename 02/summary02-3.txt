◆ 자바스크립트 소스 작성하고 실행하기

 자바스크립트 소스를 작성하는 방법은 크게 두 가지로 나눌 수 있습니다.

 1. HTML 문서 안에 자바스크립트 소스 작성하기.
 HTML 문서 안에 자바스크립트 소스를 작성하려면 HTML 태그 안에 문장을 입력 하듯이 <script> 태그 안에
 자바스크립트 소스를 작성하면 됩니다. 다음 규칙에 주의하여 <script> 태그와 함께 자바스크립트를 작성하세요.

  ① <script> 태그는 HTML 문서 어디에든 사용할 수 있습니다.
  ② <script> 태그는 한 문서 안에서 여러 번 사용될 수 있습니다.
  ③ <script> 태그가 삽입된 위치에서 소스가 실행됩니다.

 <script> 태그는 HTML 문서 어느 곳에 있어도 상관없습니다. 그러나 보통 <script> 태그는 주로 HTML 문서 내용이
 끝나는 </body> 태그 앞에 삽입합니다.

 2. 외부 스크립트 파일 연결하기.
 보통 실무에서는 자바스크립트 소스를 따로 작성하여 HTML 문서에 연결하며, 이것을 '외부 스크립트 파일을 연결한다'
 라고 합니다. 자바스크립트 소스를 따로 작성한 후 외부에서 연결하게 되면 훨씬 더 효율적으로 자바스크립트 코드를
 작성하고 관리할 수 있게 됩니다.
