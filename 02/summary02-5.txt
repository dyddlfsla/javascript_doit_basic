◆ 자바스크립트의 입력과 출력

 자바스크립트의 실습을 위해서는 앞서 배웠던 브라우저의 개발자 도구를 사용합니다.
 브라우저 화면에 마우스를 올리고 오른쪽 버튼을 클릭한 후, 메뉴 중에서 검사(Inspect)를 선택합니다.
 그러면 화면에 개발자 도구 화면이 나타나게 됩니다.

 개발자 도구 화면에서 Console 탭으로 이동합니다. 콘솔에 자바스크립트 소스를 입력하면 그 다음 줄에는
 결과 값을 표시하는데 특별히 표시할 결과값이 없을 경우 undefined 라고 표시합니다. 이것은 오류가 아니라
 출력할 내용이 없을 때 나타나는 문장입니다.

 1. 사용자 값 입력받기 - prompt() 함수
 사용자에게 값을 입력받을 때 가장 쉽게 사용할 수 있는 것이 prompt() 함수 입니다. 해당 함수를 실행하면
 브라우저는 사용자가 값을 입력할 수 있도록 작은 창을 띄워줍니다.
 prompt() 함수를 사용할 때 소괄호 안에 큰따옴표나 작은 따옴표로 원하는 문장을 감싸 넣으면 별도의 문장을
 추가할 수 있습니다. 어떤 내용을 입력해야 하는지 안내할 때 많이 사용합니다.

  ex) prompt("이름을 입력하세요");

 2. 알림 창으로 출력하기 - alert() 함수
 웹 브라우저 화면에서 간단한 알림 내용을 표시하려고 할 때 alert() 함수를 사용합니다.
 alert() 함수의 소괄호 안에 알림 내용을 입력하면 됩니다.

  ex) alert("환영합니다.");

 3. 웹 브라우저 화면에 출력하기 - document.write() 함수
 이 책의 실습을 진행하다 보면 document.write() 함수를 자주 사용하게 될 것입니다. 이 함수는 괄호 안의 내용을
 브라우저 화면에 표시하는 기능을 합니다. 보통 결과 값을 출력하는 용도로 많이 사용됩니다.

  ex) var name = prompt("이름: ");
      document.write(name + " 님, 어서오세요.");

 4. 콘솔에 출력하기 - console.log() 함수
 console.log 함수는 괄호 안의 내용을 콘솔 창에 출력합니다. document.write () 와 마찬가지로 어떤 결과 값을
 출력하고 확인하는 데 많이 사용됩니다. 차이점이라면 브라우저 화면에 출력하느냐 콘솔 창에 출력 하느냐의 차이입니다.

 5. 오류 찾아내기
 자바스크립트 소스에서 오류가 발생하여 정상적으로 동작하지 않을 경우, 어디에서 문제가 발생한 것인지 오류 내용을 찾아야
 합니다. 이 역시 브라우저의 개발자 도구를 이용하면 쉽게 확인할 수 있습니다. 오류가 발생한 브라우저 화면에서
 개발자 도구를 실행합니다. 개발자 도구의 Console 탭으로 이동하면 빨간색 글자로 표시된 오류 내용이 있습니다.
