
 ◆ Date 객체를 활용해 기념일 계산기 만들기

 1) Date 객체 간단히 알아보기
 자바스크립트에는 날짜와 시간 정보를 다루는 Date 객체가 이미 내장되어 있습니다. Date 객체를 이용하면 현재 날짜와
 시간을 알 수 있을 뿐만 아니라 특정 날짜나 시간까지 얼마나 남았는지도 계산할 수 있습니다.
 자바스크립트에서 Date 객체를 사용하려면 우선 Date 객체의 인스턴스를 만들어야 합니다.

  ① 현재 날짜 정보를 가지는 Date 객체 만들기
  이 경우, 간단히 new 예약어 다음에 Date 인스턴스를 생성하면 됩니다. 이렇게 만든 인스턴스에는 현재 날짜와 시간이
  담깁니다. new Date();

  ② 특정 날짜나 시간 정보를 가지는 Date 객체 만들기
  특정 날짜를 저장한 Date 객체를 만들고 싶다면 Date 다음와 괄호() 안에 날짜 정보를 입력합니다.
  예를 들어, '2018년 2월 25일' 날짜 정보를 가진 Date 객체를 만들고 싶다면 다음과 같이 작성합니다.
  new Date("2018-02-25");
  이렇게 만든 객체에는 지정한 날짜의 오전 9시 정보가 담깁니다.

  연도와 월까지만 지정할 수도 있습니다.
  new Date("2018-02");

  시간 정보까지 함께 지정하려면 날짜 다음에 대문자 T를 추가한 후 시간을 입력합니다.
  new Date("2018-02-25T18:00:00")
  만약 여기서 UTC 국제 표준시로 지정하려면 맨끝에 Z를 추가합니다.

 2) Date 객체의 주요 함수 알아두기
 날짜/시간 정보를 담고 있는 Date 객체를 만들었다면 Date 객체에 미리 만들어져 있는 내장 함수를 이용할 수 있습니다.
 Date 객체는 날짜와 시간 정보를 가져오는 함수와 날짜와 시간을 원하는 값으로 설정하는 함수를 가지고 있습니다.

 함수 이름 앞에 get 이나 set 이 붙어 있는데, get은 '가져오다'의 의미이고 set은 '설정하다'의 의미입니다.

  ① 날짜와 시간 정보를 가져오는 함수
   - getFullYear(): 날짜 정보에서 연도(Year)정보를 가져와 네자리 숫자로 표시합니다.
   - getMonth(): 날짜 정보에서 '월' 정보를 가져옵니다. 이때 숫자 0 ~ 11로 월을 표시합니다.
                0: 1월, 1: 2월, ...
   - getDate(): 날짜 정보에서 며칠인지 알 수 있는 '일' 정보를 가져옵니다.
   - getDay(): 날짜 정보에서 '요일'정보를 가져옵니다. 이때 요일 정보는 0 ~ 6의 숫자로 표시되는데
                0이 일요일이고 6이 토요일입니다.
   - getTime(): 1970년 1월 1일 자정 이후의 시간을 밀리초로 표시합니다.

  ② 날짜와 시간 정보를 설정하는 함수
   - setFullYear(): ...  get 이 set으로 바뀌었을 뿐 원리는 똑같다.

 Date 객체의 함수를 사용할 때 주의할 점은 getMonth()와 getDay()함수입니다. 결과값이 0부터 시작되기 때문에
 결과값에 1을 더해야 실제 '월'에 해당하는 숫자를 얻을 수 있다는 것입니다.

 3) 현재 날짜와 시간 정보 가져오기
 간단한 연습을 통해 앞서 배운 함수들을 다뤄봅니다.
 우선 Date 객체의 인스턴스를 만들고 now 변수에 저장합니다.
  var now = new Date();

 now 객체에 있는 정보 중 날짜 정보만 가져온다면 getDate() 함수를 사용합니다.
 now 객체에서 요일 정보를 가져오겠다면 getDay()를 사용합니다. 결과값이 0 ~ 6으로 표시된다는 점을 주의하세요.

 ※ getTime() 함수를 사용하면 now 객체의 시간 정보를 표시합니다. 굉장히 긴 숫자열이 결과값으로 표시되는데,
   이는 1910년 1월 1일 00:00을 기준으로 now 객체의 시간까지 시간이 얼마나 흘렀는지 밀리초(Milli Seconds)로 나타낸 것입니다.
   이렇게 값을 표시하면 특정 날짜로부터 며칠 후 혹은 며칠 전처럼 시간의 흐름을 계산할 때 편리하게 이용할 수 있습니다.
