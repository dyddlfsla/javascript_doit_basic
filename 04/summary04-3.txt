
 ◆ 명령 반복 실행하기 - for 문
 반복문은 어떤 동작을 여러 번 반복하는 데 사용합니다. 반복문을 사용하면 여러 문장을 늘어놓지 않고 간단하게
 소스를 작성할 수 있습니다.

 for 문은 초기 조건과 반복 횟수를 정확히 알고 있을 때 사용하면 좋은 반복문입니다.

  var sum = 0;
  for (var i = 1; i < 6; i++) {
    sum += i;
  }
  document.write("1부터 5까지 더하면: " + sum);

 for 문은 크게 4부분으로 나눌 수 있습니다.
 ⓐ 카운터 변수
   반복문에서 사용할 초기 변수을 설정합니다.
 ⓑ 조건식
   for 문의 블럭{} 을 실행할지 말지 결정하는 조건식입니다. 조건이 true 라면 블럭 안의 내용들이 실행됩니다.
   반대로 false 라면 for 문을 벗어납니다.
 ⓒ 실행 블록{}
   위에서 조건식이 true 일 경우 실행되는 블럭입니다.
 ⓓ 카운터 변수 조절
   실행 블록{}을 모두 실행한 뒤에 카운터 변수를 조절합니다. 보통 증감연산자 ++, -- 가 많이 사용되지만
   필요에 따라 다양한 증감식를 사용할 수 있습니다.


 ※ [ES6] for...of 문
   for 문을 사용한 반복은 보통 인덱스 값을 기준으로 반복하지만, ES6 에서는 인덱스 대신 값을 기준으로 반복합니다.
   예를 들어, for...of 문을 사용해 3개 값이 들어있는 배열을 순회할 수 있습니다.

   let season = ["봄", "여름", "가을", "겨울"];
   for (let value of season) {
    console.log(value);
   }
