
 ◆ 조건에 따라 흐름 제어하기 - if 문, if...else 문과 조건 연산자

 지금까지 작성한 소스는 위에서 아래로 순차적으로 실행되는 소스들이었습니다. 하지만 실제 프로그램에서는
 소스의 실행 순서가 결과값에 따라 뒤바뀌거나 특정 소스 부분은 건너뛰어 실행되는 등 다양한 흐름이 존재합니다.
 이렇게 프로그램에서 소스의 실행 흐름을 제어하는 명령문을 '제어문'이라고 합니다.

 1. if 문
 if 문은 아주 간단하게 작동합니다. 소괄호 안의 조건이 true 이면 중괄호 블럭{} 안의 내용을 실행하고
 false 이면 중괄호 블럭{} 을 무시하고 진행합니다.

 2. if-else 문
 if 문에서 조건식이 false 일때 중괄호 블럭{}을 무시하는 것에 그치지 않고 특정한 코드를 실행하고 싶을 때는
 else 구문을 추가하여 사용합니다.

 3. 조건 연산자 - ?:
 조건식이 하나이고 true 일 때와 false 일 때 실행해야할 명령도 각각 하나뿐이라면 if-else 문 보다는
 조건 연산자를 사용하는 것이 더 좋을 수 있습니다. 조건 연산자는 기호 ?와 기호 :로 이루어져 있습니다.

  (조건식) ? (true 일때 실행되는 명령문) : (false 일때 실행되는 명령문)

 4. truthy 값과 falsy 값
 논리형 자료 값은 true 와 false 뿐입니다. 그런데 자바스크립트에서는 'true 로 인정할 수 있는 값'과
 'false 로 인정할 수 있는 값'이라는게 있습니다. 이러한 값을 가리켜 'truthy 하다', 'falsy 하다'라고 표현합니다.
 다음은 자바스크립트에서 falsy 하게 인정하는 값들입니다.

  ① 0
  ② ""
  ③ NaN (Not a Number, 변수를 선언만 하고 할당하지 않은 상태에서 연산 했을 때 저장되는 값)
  ④ undefined
  ⑤ null

  var input = prompt("이름을 입력하세요.");
  if (input) {
   alert("이름을 입력했습니다." + input);
  } else {
    alert("이름을 입력하지 않았습니다.");
  }

 위에 나열한 5개의 값이 조건식에 대입되면 해당 조건식은 false 를 반환합니다. 반대로 이 falsy 값을 제외한
 나머지 값은 자바스크립트에서 true 로 인정하는 truthy 값입니다.
 truthy 값과 falsy 값은 조건을 확인할 때 유용하게 사용할 수 있습니다.
 프롬프트 창에 아무것도 입력하지 않았거나("") [취소]버튼을 눌렀을 때(null) 조건식은 자동으로 false 를 반환하며
 반대로 무엇이라도 입력하게 된다면 truthy 값이 되어 true 를 반환합니다.
