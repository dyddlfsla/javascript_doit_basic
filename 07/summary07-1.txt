
 ◆ Array 객체란?
 배열(Array)은 자바스크립트에서 자주 사용하는 자료형이기 때문에 미리 자바스크립트 안에 프로토타입 객체로 만들어져 있습니다.
 그리고 그 안에는 배열에 활용하는 많은 속성과 함수가 미리 정의되어 있습니다.

 자바스크립트에서는 여러 개의 항목을 하나의 변수에 저장해야 할 때 '배열(Array)'을 사용합니다.

 1) Array 객체로 배열 만들기
 앞에서 리터럴 표기법을 활용한 배열 만들기를 알아보았습니다. 여기에서는 객체의 인스턴스를 만드는 방법으로
 배열을 만들어 봅니다. 자바스크립트에서는 배열을 쉽게 만들고 다룰 수 있게 미리 Array 객체가 만들어져 있습니다.

 다음 소스는 Array 객체의 인스턴스를 만든 후 myArray 변수에 저장한 것입니다.

 var myArray = new Array(); // Array 객체의 인스턴스를 생성.

 빈 괄호를 사용하면 배열 요소의 개수가 고정되지 않기 때문에 myArray 배열에 얼마든지 많은 자료를 저장할 수 있습니다.

 초기값이 있는 배열이라면 앞에서 배운 바와 같이 리터럴을 이용한 방법으로 배열을 만들수도 있고,
 Array 객체를 사용해서 만들 수도 있습니다.

 var numbers = ["one", "two", "three", "four"]; // 리터럴 표기법을 활용한 배열 생성
 var numbers = new Array("one", "two", "three", "four", "five"); // Array 객체를 활용한 배열 생성

 2) 배열에서 for 문 사용하기
 배열을 생성한 후 해당 배열의 내용을 for 문을 통해 출력해 볼 수 있습니다. 그리고 배열에 몇 개의 요소가 저장되어 있는지
 확인하려면 Array 객체의 length 속성을 사용합니다.

 var seasons = ["봄", "여름", "가을", "겨울"];
 seasons.length;
 4

 seasons 배열에 4개의 요소가 있기 때문에 seasons[0] 부터 seasons[1], seasons[2], seasons[3] 값을 가져오려면
 for 문을 사용해서 0부터 3까지 반복하면 됩니다.

 for ( i = 0; i < seasons.length; i++) {
     console.log(seasons[i]);
 }
