
 ◆ 폼 요소에 접근하는 여러 가지 방법

 웹 사이트의 폼 요소에는 사용자가 내용을 입력하는 텍스트 필드를 비롯해 라디오 버튼이나 체크 상자처럼
 누를 수 있는 요소도 있고, 선택(select) 목록처럼 여러 항목에서 원하는 것을 선택해서 사용하는 요소도 있습니다.

 이런 요소를 자바스크립트 프로그램으로 제어하려면 먼저 요소에 접근해야겠죠? 폼 요소에 접근할 수 있는
 여러 방법들을 알아봅니다.


 1) id 값이나 class 값을 사용해 폼 요소에 접근하기
 id 값이나 class 값을 사용해 폼 요소에 접근하는 방법은 DOM 의 다른 요소에 접근하는 것과 같습니다.
 querySelector() 함수나 querySelectorAll() 함수를 사용해서 특정 id 값이나 class 값을 가진 요소에 접근할 수 있습니다.
 id 값을 사용할 떄는 폼 요소 하나에만 접근하고, class 값을 사용할 때는 해당 값을 가진 모든 폼 요소를 가져와 배열 형태로 저장합니다.

 document.querySelector("#billingName");

 텍스트 필드에 있는 값을 가져오기 위해서는 텍스트 필드에 접근하는 소스 뒤에 value 속성을 가져오면 됩니다.

 document.querySelector("#billingName").value;

 2) name 값을 사용해 폼 요소에 접근하기

 폼 요소에 id 나 class 속성이 없고 name 속성만 있다면 name 식별자를 이용해 폼 요소에 접근할 수도 있습니다.
 id 나 class 속성은 웹 개발에 CSS 를 사용하기 시작하면서부터 등장했지만, name 속성은 자바스크립트에 폼 요소를 구별하고 접근할 수 있도록
 HTML 초기부터 사용하던 방법입니다. 직접 폼을 작성한다면 id 나 class 선택자를 사용하는 것이 낫지만, 다른 사람이 작성해 놓은
 폼 소스에  name 속성만 있다면 이 방법을 사용해야 합니다.

 이 방법을 사용하려면 <form> 태그에 name 속성이 지정되어 있어야 하고 <form> 태그 안에 포함된 폼 요소에도 name 속성이 있어야 합니다.
 name 속성을 사용해 폼 요소에 접근하려면 form 의 name 값부터 폼 요소의 name 값까지 계층을 따라 하나씩 지정해 줍니다.

 ※ 폼 요소에서 사용하는 name 값은 주로 id 값과 동일하게 지정합니다.

 폼 안에 있는 요소에 접근하려면 <form> 의 name 값과 해당 요소의 name 값을 함께 사용하면 됩니다.

 ① document.ship.shippingInfo

 또는 다음과 같이 할수도 있습니다.

 ② document.forms["ship"].elements["shippingName"]

 name = "shippingName" 인 요소의 소스를 선택할 수 있게 됩니다. 그리고 다시 이 소스 뒤에 value 를 붙이게 되면
 앞서와 마찬가지로 해당 요소에 입력된 값을 가져올 수 있습니다.

 document.ship.shippingName.value;

 id 식별자를 통해 접근하는 방법은 접근할 요소에 id 속성이 지정되어 있다면 쉽게 접근할 수 있지만, name 속성을 사용해 접근하려면
 <form> 태그뿐만 아니라 접근하려는 폼 요소에 모두 name 속성이 지정되어 있는지 확인해야 합니다.

 3) 폼 배열을 사용해 폼 요소에 접근하기

 이번에는 폼 배열을 사용해 폼 요소에 접근해 보겠습니다. document 의 속성 중 forms 속성은 문서 안에 있는 <form> 태그를 모두 가져와 배열 형태로 반환합니다.
 이 방법은 폼 요소에 id 나 class 가 없고 name 값 마저 없을 때 사용할 수 있습니다.

 폼 배열을 이용하는 방법은 웹 문서에 <form> 태그나 몇 개가 있는지 알고 있어야 하고 <form> 요소에 폼 요소가 많을 경우 원하는 요소에 접근하기가 어려워
 실무에서 자주 사용하지는 않습니다.

 ① document.forms

 다음과 같이 forms 를 사용하면 웹 문서에 존재하는 <form> 태그 갯수만큼 HTMLCollection 배열에 담겨 옵니다.

 그리고 <form> 태그 안에 포함된 폼 요소에 접근하기 위해 elements 속성을 사용합니다. 마찬가지로 해당 <form> 안에 있는 폼 요소들을
 HTMLFormControlsCollection 이름의 배열로 가져옵니다.

 document.forms[0].elements[1].value;

 해당 문장은 forms 배열에서 0번째 form 배열에 접근한뒤 다시 이 배열의 인덱스 1번 자리에 있는 폼 요소에 접근합니다.
 또 뒤에 value; 를 붙여 입력된 값을 받아 올수도 있습니다.
