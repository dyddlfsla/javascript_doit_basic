
 ◆ 변수란 무엇일까?
 자바스크립트에서 변수는 변하는 값을 저장하는데 사용합니다. 예를 들어 누군가의 생일을 데이터로 저장한다면
 A 의 생일과 B 의 생일은 각각 다르게 저장되어야 합니다. 이처럼 상황에 따라 다른 값을 입력 받아야 할 때
 사용하는 것이 변수입니다.

 1. 수식으로 알아보는 상수와 변수

 나이를 계산해주는 프로그램은 어떻게 나이를 계산했을까요?
 다음은 나이를 구하는 과정을 식으로 표현한 것입니다.

 age = currentYear - birthYear + 1

 이 수식의 맨 끝에 있는 1은 변하지 않는 값입니다. 이런 값을 프로그래밍에서는 '상수(Constant)'라고 부릅니다.
 반면에 나머지 값들 (올해 연도, 태어난 연도, 나이)은 변할 수 있는 값이죠. 이런 값을 변수(variable)이라고 합니다.

 2. 변수를 선언하는 세 가지 규칙
 변수에는 컴퓨터가 구별할 수 있도록 이름을 붙여야 합니다. 이를 '변수를 선언한다'라고 합니다. 변수 선언은 아주 간단합니다.
 다음처럼 키워드 var 뒤에 변수 이름을 적어주면 됩니다.

  var apple;

 그런데 변수를 선언할 때에는 지켜야할 몇 가지 규칙이 있습니다.

 ⑴ 변수의 이름은 의미있게 짓는다.
 변수의 이름을 아무 의미 없이 지으면 안됩니다. a, b 같은 변수는 나중에 왜 만들었는지 알기 어렵습니다.
 예를 들어 위의 예처럼 올해 연도를 저장하는 변수라면 currentYear, 태어난 연도라면 birthYear 라고 지으면
 해당 변수가 어떤 값을 저장하는지 더 쉽게 알아볼 수 있습니다.

 ⑵ 여러 단어가 조합된 변수이름은 낙타표기법(camel case)를 지켜 작성한다.
 여러 단어를 연결하여 변수 이름을 지을 때에는 첫 글자는 문자 또는 $, _ 로 시작해야 하며 두 번째 단어부터는
 대문자로 시작하는 표기를 많이 사용합니다. 이런 규칙을 낙타 표기법이라고 하는데 변수 표기법에 강제된 규칙은 없지만
 개발자들 사이에서 오래전부터 암묵적으로 지켜오는 규칙이므로 꼭 지키는 것이 좋습니다.

 ⑶ 변수 이름으로 사용할 수 없는 단어들도 있다.
 true, false, if 등 자바스크립트에서 키워드로 사용되는 단어들은 변수 이름으로 사용할 수 없습니다.

 3. 변수에 값이나 식 저장하기
 변수에 값을 저장하려면 변수에 대입연산자(=)를 붙인 뒤 저장하고 싶은 값을 입력하면 됩니다.

  var apple = "yummy";
  var banana;
  banana = "long";

 이때 변수를 선언하면서 동시에 값을 저장할 수도 있고 변수를 선언만 하고 값은 나중에 저장할 수도 있습니다.

  ※ [ES6] let 과 const 에 대하여
  자바스크립트 ES6 버전에서부터는 변수를 선언할때 var 예약어 이외에 let 과 const 를 사용할 수 있습니다.
  let 으로 선언한 변수는 블록({}으로 선언된 범위)을 벗어나면 사용할 수 없습니다. 그리고 const 는
  상수값을 선언할 때 사용됩니다.
